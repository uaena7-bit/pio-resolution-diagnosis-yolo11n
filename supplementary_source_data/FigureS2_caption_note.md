# Figure S2 caption note

Recommended caption text for the supplementary materials:

Figure S2. Strict global-first diagnostic AP by (a) target scale group and (b) image density group. Strict diagnostic AP preserves the global ranked prediction list and global matching results before attributing matched true positives to predefined groups. Therefore, it measures subgroup contributions under the full-test-set operating order and is intended for within-protocol diagnostic comparison only. The values are not directly comparable with ordinary subgroup AP values in Figs. 3 and 4 or with standard global AP.
